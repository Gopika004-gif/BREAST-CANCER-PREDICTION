# 📘 Breast Cancer Prediction using Random Forest

## 📌 Introduction
Breast cancer is one of the most common and life-threatening diseases affecting women worldwide. Early detection plays a crucial role in improving survival rates and treatment success.

This project uses **Machine Learning** to predict whether a tumor is **Malignant (cancerous)** or **Benign (non-cancerous)** based on medical diagnostic data. The model helps support early diagnosis and clinical decision-making.

---

## 🎯 Objective
The main objectives of this project are:

- Build a machine learning model for breast cancer prediction  
- Analyze and understand dataset features  
- Evaluate model performance using accuracy and confusion matrix  

---

## 📊 Dataset

- **Dataset:** Breast Cancer Dataset (from `sklearn`)

### Features:
- Mean radius  
- Texture  
- Perimeter  
- Area  
- Smoothness, etc.  

### Target:
- `0 → Malignant (Cancerous)`  
- `1 → Benign (Non-cancerous)`
